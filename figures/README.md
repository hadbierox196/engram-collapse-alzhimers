## Figure 1 — Retrieval Fidelity Trajectories under Simulated Braak-Stage Degradation

**File:** `fig1_fidelity_curves.png / .svg`

Shows how mean cosine retrieval fidelity declines for each of the four memory classes as the synaptic degradation parameter r increases from 0 (healthy) to 1 (severe). The x-axis is the degradation ratio r; the y-axis is mean cosine fidelity averaged across N ∈ {500, 1000, 2000}, three overlap levels, five noise levels, and three degradation rates (n = 135 configurations per class). Shaded bands are 95% confidence intervals.

Background shading marks the three Braak stage zones:
- Blue (r = 0.00–0.33): Stage 1, Transentorhinal
- Yellow (r = 0.33–0.66): Stage 2, Limbic
- Red (r = 0.66–1.00): Stage 3, Isocortical

The dashed horizontal line at fidelity = 0.50 is the collapse threshold. Vertical dotted lines mark each class's mean r\* (the degradation ratio at which fidelity first crosses 0.50).

**What it shows:** All four classes maintain near-perfect plateau fidelity (≥ 0.999) before their collapse point, then drop sharply. The failure sequence is strictly ordered:

| Memory Class | Mean r\* | Range |
|---|---|---|
| weak\_remote    | 0.756 | 0.700–0.800 |
| weak\_dense     | 0.883 | 0.825–0.900 |
| strong\_recent  | 0.964 | 0.925–1.000 |
| strong\_salient | 0.983 | 0.950–1.000 |

This ordering is robust across all 540 parameter configurations. Main effect of memory class: F(3,536) = 13,187.93, p < 2.2×10⁻¹⁶, η² = 0.987.

## Figure 2 — Parameter Sensitivity across 135 Simulation Configurations

**File:** `fig2_sensitivity_panel.png / .svg`

A 2×2 panel examining how model outputs respond to variation in network size (N), degradation rate, and retrieval noise. All panels use data from `week5_full_results.csv`. Error bars are 95% CIs.

### Panel A — Post-Collapse Floor by Memory Class

Bar chart of mean fidelity at r ≥ 0.90 (after all classes have collapsed). Shows how much residual retrieval capacity each class retains at severe degradation.

| Memory Class | Post-collapse floor |
|---|---|
| weak\_remote    | 0.006 |
| weak\_dense     | 0.020 |
| strong\_recent  | 0.442 |
| strong\_salient | 0.548 |

Weak classes collapse to near-zero residual fidelity. Strong classes, particularly strong\_salient, retain partial retrieval capacity even at maximum degradation — consistent with the clinical observation that procedural and emotionally salient memories are most resistant in late-stage AD.

### Panel B — AUC by Network Size and Memory Class

Grouped bar chart of area under the fidelity curve, broken down by N ∈ {500, 1000, 2000}. AUC captures total memory capacity across the full degradation trajectory.

| Class | N=500 | N=1000 | N=2000 |
|---|---|---|---|
| weak\_remote    | 0.694 | 0.753 | 0.778 |
| weak\_dense     | 0.836 | 0.866 | 0.873 |
| strong\_recent  | 0.938 | 0.944 | 0.944 |
| strong\_salient | 0.945 | 0.960 | 0.964 |

Interaction effect F(6,528) = 131.92, p = 1.23×10⁻¹⁰¹, η² = 0.600. Network size has a strongly class-differentiated effect — see Figure 4 for a dedicated analysis.

### Panel C — Collapse Sharpness by Degradation Rate

Line plot of transition width (the r-span from fidelity = 0.90 to fidelity = 0.10) at each of the three degradation rates (slow = 0.025, medium = 0.050, fast = 0.100 per step). A wider transition means a more gradual collapse; a narrower transition means an abrupt, cliff-like failure.

Faster degradation produces sharper collapses across all classes. weak\_remote shows the widest transition width range (0.036–0.114), indicating it is most sensitive to how quickly degradation accumulates. Pearson r = +0.670 between step size and transition width (p < 10⁻⁷¹).

### Panel D — Noise Effect on Pre-Collapse Plateau (weak\_remote)

Line plot of pre-collapse plateau height for weak\_remote as retrieval noise σ increases from 0.10 to 0.50. Other classes are shown as flat reference lines at fidelity ≈ 1.0.

| σ | weak\_remote plateau |
|---|---|
| 0.1 | 0.99999 |
| 0.2 | 0.99999 |
| 0.3 | 0.99978 |
| 0.4 | 0.99890 |
| 0.5 | 0.99593 |

The absolute effect is small (max Δ = 0.4% at σ = 0.5) but statistically reliable: t(52) = 4.06, p_Bonferroni = 0.003. Strong classes are unaffected. This confirms that weak\_remote memories are vulnerable not only to earlier collapse but also to mild plateau erosion under noisy retrieval — even in the healthy baseline regime.

## Figure 3 — Clinical Validation: Model Predictions vs ADNI Empirical Decline

**File:** `fig3_slopegraph_adni.png / .svg`

Two-panel figure comparing the model's predicted memory failure sequence against longitudinal cognitive decline data from 191 MCI-to-AD converters in the ADNI cohort (DXSUM + NEUROBAT tables). Cognitive scores are z-standardised; slopes are estimated per subject via OLS over a ±36-month window centred on conversion.

### Panel A — Rank Alignment Slopegraph

Side-by-side ranking of three cognitive domains by steepness of decline. Left side shows model-predicted rank; right side shows empirical rank from ADNI.

| Domain | Model rank | Empirical rank | Match |
|---|---|---|---|
| Episodic Memory (RAVLT Delayed Recall) | 1 | 1 | ✓ |
| Working Memory (Digit Span) | 3 | 2 | ✗ |
| Semantic Memory (BNT) | 2 | 3 | ✗ |

Episodic memory ranks first in both. Working and semantic memory are transposed relative to the model prediction. Spearman ρ = 0.50, p = 0.667 (n = 3 domains). This is not statistically significant — at n = 3, even a perfect ordering only achieves p_min = 0.167, making rank correlation an underpowered test at this domain count.

### Panel B — Longitudinal Decline Trajectories

Line plot showing mean z-score trajectories for each cognitive domain across months −36 to +12 relative to AD conversion (month 0). Shaded bands are ±1 SEM across subjects. All three domains show significant pre-conversion decline.

| Domain | Slope (z/month) | 95% CI | p |
|---|---|---|---|
| Episodic Memory | −0.02487 | [−0.034, −0.016] | < 0.001 |
| Working Memory  | −0.02420 | [−0.034, −0.015] | < 0.001 |
| Semantic Memory | −0.02184 | [−0.030, −0.014] | < 0.001 |

The three slopes are close in magnitude (range = 0.003 z/month), which partly explains why the rank ordering is sensitive to small estimation noise. The episodic–working transposition in the model may also reflect the coarse mapping between cognitive test domains and Hopfield memory classes.

## Figure 4 — Network Size Preferentially Benefits the Most Vulnerable Memory Class

**File:** `fig4_auc_by_N.png / .svg`

Line plot of mean AUC across N ∈ {500, 1000, 2000} for all four memory classes. Points are slightly offset horizontally by class for readability; error bars are 95% CIs. Percentage gain labels show the relative AUC improvement from N = 500 to N = 2000.

| Class | AUC gain (N=500→2000) |
|---|---|
| weak\_remote    | +12.0% |
| weak\_dense     | +4.4%  |
| strong\_recent  | +0.7%  |
| strong\_salient | +2.1%  |

The effect of network size is strongly class-differentiated. weak\_remote benefits the most (+12.0%), while strong\_recent benefits least (+0.7%). This is consistent with the theoretical expectation that larger networks provide more orthogonal representational space, which disproportionately helps low-sparsity, low-amplitude patterns (weak\_remote) that are most susceptible to inter-pattern interference.

Main effect of N: F(2,528) = 655.00, p = 9.73×10⁻¹⁴⁴, η² = 0.713.
